Compile MYSQL++ in VS2010

Tue 11 February 2014
By icylord

In Computer Vision.

tags: Computer VisionPython

1、到http://tangentsoft.net/mysql++/下载mysql++-3.2.1

链接:MYSQL++

2、解压缩文件

3、进入mysql++-3.2.1\vc2008目录

4、打开mysql++. sln，让VS2010做自动转换，因为VS2010的工程配置目录改变了，因此VS2008的一些配置会丢失，因此需要手动添加mysql c connector的目录。同时只需要编译mysqlpp的lib，因此其他例子程序就编译了，所以下文只修改mysqlpp的工程目录。 5、默认的工程是编译x64的库，非常好编译，因此下文主要处理32位的情况。

6、首先点击vs2010上方的x64，选择Configuration Manager，在active sulution platform选择new一个win32的配置，并复制x64的配置。然后把mysqlpp的platform改成win32。

7、接着在工程中additional include directories添加mysql connector c的include路径。并且在preprocessor definitions 添加WIN32;_USRDLL;DLL_EXPORTS;_DEBUG;UNICODE;_UNICODE;MYSQLPP_MAKING_DLL;HAVE_MYSQL_SSL_SET;%(PreprocessorDefinitions)，这段宏定义可以在x64的相同位置找到。

8、接着在mysqlpp工程中修改configuration type为 Dynamic Library；

9、在mysqlpp工程中Linker->General->Additional Library Directories中添加C:\Program Files (x86)\MySQL\MySQL Connector C 6.1\lib，然后在Linker->Input->Additional Dependencies中添加libmysql.lib和wsock32.lib；

10、最后一步需要修改工程的Runtime Library为MT的，进入C/C++->Code Generation->Runtime Library，修改为/MDd

11、右键Solution Explorer点击mysqlpp点击build，即可生成mysqlpp.dll和mysqlpp.lib.

12、Release版本的跟Debug类似，后续就不进行描述了。
